## 📖 What's Inside

### ⚙️ Operating Systems

> *"Bridging the gap between hardware and you"*

| Topic | Key Points |
| :--- | :--- |
| **OS Types** | Single-tasking (MS-DOS), Multiprogramming, Multiprocessing |
| **Process Management** | RAM allocation, Process states, Context switching |
| **Memory Techniques** | Segmentation, Paging, Contiguous allocation |
| **Threads** | Lightweight execution units for parallelism |

### 💠 Object Oriented Programming (C++)

> *"Real-world modeling through code"*

| Topic | Key Points |
| :--- | :--- |
| **Paradigms** | Procedural vs OOP — Data as the critical element |
| **Encapsulation** | Data hiding — functions tied to data |
| **Advanced Concepts** | Inline functions, Function overloading, Operator overloading |
| **Static Members** | Class-level variables and functions |
| **Virtual Functions** | Runtime polymorphism & Pure virtual (Abstract) classes |
| **Namespaces** | Avoiding naming conflicts like a pro |

### 🗄️ Database Systems

> *"Accessing and modifying data—beautifully organized"*

| Topic | Key Points |
| :--- | :--- |
| **RDBMS** | Data stored through relations (tables) |
| **Clustered Index** | Data ordered by primary key — fast reads |
| **Non-Clustered Index** | Secondary indexing — logical ordering |
| **Hashing** | Direct key-to-location mapping for O(1) lookups |
